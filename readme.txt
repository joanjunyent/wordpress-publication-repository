== Description ==

It allows to manage a publication repository and exposes metadata in order to be indexed by Google Scholar


== Installation ==

1. Install and activate it through the plugin management screen.
2. Start adding publications!

In the single pages the needed metadata is shown in order to get the publication indexed by Google Scholar.
